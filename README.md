# CU Today - Event Information App

## 📌 Overview
CU Today is a **Flutter and Firebase-powered** mobile application designed to provide students with real-time updates on **tech events, cultural events, seminars, hackathons, fests, and sports events** happening on campus. The app ensures that students stay informed and engaged in university activities.

## 🚀 Features
- 📅 **Event Listing** - View all upcoming and past events categorized by type.
- 🔎 **Search & Filters** - Find events based on categories like tech, cultural, sports, and date.
- 📌 **Event Details** - View complete event information including location, time, and description.
- 🔔 **Push Notifications** - Get reminders and updates about new and upcoming events.
- 📍 **Favorites & Reminders** - Save favorite events and set reminders.
- 🔐 **User Authentication** - Secure login via **Google Sign-In** and **Email/Password**.
- 🛠 **Admin Panel** - Allows admins to **Add/Edit/Delete** events.
- 🌙 **Dark Mode Support** - Toggle between light and dark themes.

## 🏗 Tech Stack
- **Frontend**: Flutter (Dart)
- **Backend**: Firebase (Firestore, Authentication, Cloud Messaging)
- **State Management**: Provider/Riverpod
- **IDE**: Android Studio
- **Design**: Figma

## 🔧 Getting Started
### Prerequisites
Before running the project, ensure you have:
- **Flutter & Dart SDK** installed.
- **Android Studio** or **VS Code** with the Flutter plugin.
- A **Firebase project** setup with `google-services.json` for Android & `GoogleService-Info.plist` for iOS.

### Setting Up Firebase
1. Go to **Firebase Console** and create a new project.
2. Enable **Firestore Database** and **Authentication**.
3. Download `google-services.json` and place it inside `android/app/`.
4. For iOS, download `GoogleService-Info.plist` and place it in `ios/Runner/`.
